# Repository Rules

This repository tracks my progress through Striver's SDE Sheet.

## README Rules

When modifying README.md:

- NEVER rewrite existing sections.
- NEVER change the repository description.
- NEVER change the About Me section.
- NEVER change badges.
- NEVER change formatting or styling.
- NEVER reorder solved problems.
- NEVER remove existing rows.

Only do the following:

1. Append new solved problems to the Progress Tracker table.
2. Use local repository links only.
3. Keep numbering consistent with existing entries.
4. Notes should default to "-" unless explicitly provided.
5. Status should be "✅".
6. Preserve the exact table format already present.

## Folder Rules

Problems may be stored in folders such as:

- 01Arrays
- 02LinkedList
- 03Greedy
- etc.

Always link to the correct local file.

Example:

[View Code](./02LinkedList/17_RemoveNthNode.cpp)

## Important

This repository is intentionally minimal.

Do not:
- Add statistics sections.
- Add progress bars.
- Add learning summaries.
- Add extra badges.
- Add GitHub widgets.
- Add animations.
- Redesign the README.

Only append new entries.

If a change would modify more than 10 lines outside the Progress Tracker table,
STOP and ask for confirmation.
